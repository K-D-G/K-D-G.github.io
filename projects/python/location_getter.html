Projects:Location Getter

This project is about getting your location using python.
To do this I will be using an api as well as a module called requests as well as JSON; which is part of the standard library

The code


        import requests
        import json

        sess=requests.Session()
        data=sess.get('http://freegeoip.net/json')
        data=json.loads(data.text)
        print(data.text)

Explanation: The code imports the 2 libraries in use then creates a session (using requests). Then it makes a GET request to the website. Then the JSON library loads it as a string and then the data is printed
